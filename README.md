**Comando para clonar el repositorio**
```shell
git clone -b develop https://github.com/LeonelSistemas5/SistemaCPv1.git
```

**Comandos de inicio:**
```shell
npm install
```
```shell
npm run dev
```
```shell
composer install
```
```shell
composer update
```
```shell
php artisan migrate --seed
```
```shell
php artisan storage:link
```
```shell
php artisan serve
```
