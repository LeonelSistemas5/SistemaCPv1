**Comando para clonar el repositorio**
```shell
git clone https://github.com/LeonelSistemas5/SistemaCPv1.git
```

**Comandos de inicio:**
```shell
composer install
```
```shell
npm install
```
```shell
php artisan key:generate
```
```shell
php artisan storage:link
```
```shell
php artisan migrate --seed
```
```shell
php artisan migrate:fresh --seed
```
```shell
npm run dev
```
```shell
php artisan serve
```
